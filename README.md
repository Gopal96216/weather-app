# 🌦️ Weather App  
A simple web-based application to check real-time weather updates using OpenWeather API.  

## 🚀 Features  
✔️ Search weather by city  
✔️ Displays Temperature, Humidity, and Wind Speed  
✔️ User-friendly and Responsive UI  

## 📌 Technologies Used  
- HTML  
- CSS  
- JavaScript  
- OpenWeather API
